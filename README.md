# React + Vite

## Как запустить приложение
1. Установить node.js с менеджером пакетов npm
2. Запустить консоль в директории проекта
3. Ввести в консоли команду ```npm install yarn```
4. Установить vite c помощью команды ```yarn add vite```
5. Запустить проект с помощью команды ```yarn dev```
